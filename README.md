# Tiny OpenGL Example
> A tiny OpenGL example of a bouncing textured triangle.


## Dependencies
> Install these:
```bash
    libglfw3-dev libglew-dev
```
> Also download and compile & install truetype2:
```
git clone git://git.sv.nongnu.org/freetype/freetype2.git
```

## Build & Run
> Run:
```bash
make && ./a.out
```
